====================================
Immersive Armors - CCOR Patch Notes
====================================
This file patches Immersive Armors by Hothtrooper44 (https://www.nexusmods.com/skyrimspecialedition/mods/3479) for improved consistency with changes made by Complete Crafting Overhaul Remastered.

This is a .esp / ESL flagged file and won't take up a spot in your load order.

INSTALLATION
Both of these mods are required in order to use this file.
Load Immersive Armors after Complete Crafting Overhaul Remastered.

CHANGES
- Added crafting menu filters and category conditions from CCOR SE.
- Moved any breakdown recipes that break items down into leather or hide to the Tanning Rack menu.
- Added WAF_PerkFists keywords to light gauntlets.
- WAF_ProtectiveClothing keywords were added to unarmored robes to allow them to work with the Protective Clothing option from Armor & Clothing Extension (https://www.nexusmods.com/skyrimspecialedition/mods/19002).
- Barbarian Hero, Einherjar, Falkreath Armor, Spellbinder, Vanguard Plate, Witchplate, and a number of Shield recipes were changed to use Galatite. 
- The following heavy armor sets now require the Advanced Heavy smithing perk instead of the Advanced Light perk:
	- Falkreath Armor
	- Hedge Knight
	- Vanguard Plate
	- Einherjar Plate
	- Primitive Nord
	- Heavy Jorrvaskr Shield
	- Spellbinder armor
