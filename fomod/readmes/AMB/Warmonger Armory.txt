==================================
Warmonger Armory - AMB Patch Notes
==================================
This file modifies Warmonger Armory - Unofficial Fixed SSE Port by Batmanna and Duckzilla (https://www.nexusmods.com/skyrimspecialedition/mods/17809) for improved consistency with aMidianBorn Content Addon.
This is a .esp / ESL flagged file and won't take up a spot in your load order.

INSTALLATION
Both of these mods are required in order to use this file.
Load this patch after WarmongerArmory_WACCF_Patch.esp or WarmongerArmory_CCOR_Patch.esp.
